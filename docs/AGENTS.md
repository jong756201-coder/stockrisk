# 프로젝트 미션
이 프로젝트는 고위험 모멘텀 주식을 위한 근거 기반 대시보드다.
이 프로젝트는 추천 엔진이 아니며, 매수/매도 조언을 생성해서는 안 된다.

# 제품 우선순위
1. 빠른 반복 방문 유틸리티
2. 명확한 근거 제시
3. 미래 확장이 가능한 구조

# MVP 범위
- 공개 Movers Feed
- 종목 상세 페이지
- 구조화된 근거 패널
- 최소한의 내부 관리자 검수 도구
- 기본 로그인 및 관심종목(Watchlist) 기능
- 커뮤니티 없음
- 댓글 없음
- 개인화된 매매 시그널 없음
- 포트폴리오 조언 없음
- 해외주식 급등주/테마주만을 다룬다.

# 기술 원칙
- 나중에 큰 재작성을 유발하는 지름길보다 모듈형 구조를 우선한다
- 성급한 과설계는 피한다
- 공개 앱, 관리자 앱, 백그라운드 작업을 논리적으로 분리한다
- 데이터 추출은 schema 기반으로 설계한다
- 리스크 관련 UI는 모두 근거와 source 링크를 가져야 한다

# UX 원칙
- 사용자는 30~60초 안에 한 종목을 이해할 수 있어야 한다
- 불투명한 점수보다 근거 카드 중심으로 보여준다
- 무엇이 공개되었는지, 무엇이 빠졌는지, 유사 사례가 어땠는지를 보여준다

# 안전 / 법적 제약
- 어떤 종목도 fraud, scam, manipulation, safe라고 라벨링하지 마라
- 직접적인 매수/매도 추천을 하지 마라
- disclosed, missing, historical analogues, listing status 같은 중립적 표현을 사용하라

# 엔지니어링 워크플로우
- 큰 작업은 먼저 계획을 세우고 승인을 기다려라
- 구현 전에 schema와 routes를 먼저 제안하라
- 커밋 또는 작업 결과는 작고 검토 가능하게 유지하라
- 모든 가정을 문서화하라
